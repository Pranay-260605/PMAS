# Usecase Diagram

![](Usecase_diagram.png)

1. Actors
1.1 Human Actors

User
Responsible for monitoring machine health and viewing alerts generated by the system.

Technician
Responsible for logging into the system, managing machines, monitoring machine health, and creating maintenance tickets.

System Administrator
Manages user access, registers technicians, manages machines, and views or updates maintenance tickets to ensure smooth system operation.

1.2 External System Actors

System (Predictive Maintenance Automation System)
Performs automated operations such as collecting sensor data, preprocessing data, analyzing machine health, detecting anomalies, predicting failure risks, and generating alerts.

Sensor Data Source
Provides real-time and historical sensor data from machines to the system for analysis.

Alert Notification System
Delivers alerts and notifications to users when potential machine failures or anomalies are detected.
